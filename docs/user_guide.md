# Справка пользователя

## Назначение
Программа «Калькулятор» предназначена для выполнения арифметических операций над вещественными числами и перевода натурального числа в восьмеричную систему счисления.

## Запуск
1. Установить Python 3.
2. Запустить файл `calculator.py`.

## Основные действия
- Ввод выражения выполняется кнопками интерфейса.
- Для вычисления результата нажать `=`.
- Для очистки поля ввода нажать `C`.
- Для перевода натурального числа в восьмеричную систему ввести число и нажать `Oct`.

## Ошибки
Программа выводит сообщения об ошибках при делении на ноль, некорректном выражении или неверном числе для перевода.
