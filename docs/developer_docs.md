# Документация программиста

## Структура проекта
- `calculator.py` — основной файл приложения.
- `README.md` — краткое описание проекта.
- `docs/user_guide.md` — справка пользователя.
- `docs/developer_docs.md` — документация программиста.

## Используемые библиотеки
- `tkinter` — создание графического интерфейса.
- `messagebox` — вывод сообщений об ошибках и результата перевода.

## Основные функции
- `add_symbol(symbol)` — добавляет символ в поле ввода.
- `calculate()` — вычисляет введенное выражение.
- `clear()` — очищает поле ввода.
- `to_octal()` — переводит натуральное число в восьмеричную систему.

## Обработка исключений
В функции `calculate()` предусмотрена обработка деления на ноль и некорректных выражений. В функции `to_octal()` проверяется корректность натурального числа.
